Guix is a declarative package manager for GNU/Linux (and other kernels like Hurd) that can be used as a stand-alone distribution too. It is similar to Nix, but as a main killing feature, its configuration is pure Guile.

According to (b9hummingbird 2024), Guix does not work on Android (yet).
