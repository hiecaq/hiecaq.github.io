Meow

2025-04-09
Forward links: related:Emacs Backward links: id:Noun-Verb for Evil

meow is a new model-editing package for Emacs that

  • Encourages and is designed around Noun-verb text editing (e.g. select a quoted string with visual feed-back, then delete it)
  • Provide some special basic movements that can be repeated with a number or reversed with a negative prefix
  • Has a God-mode-like state called KEYPAD mode
  • Has a layer build upon Keyboard Macro called BEACON

Reference

Built with Emacs 30.0.92 (Org mode 9.7.25)
